# add husky

1. 安装 husky

```js
pnpm i husky -D
```

2. package.json

```json
{
  "scripts": {
    // 会在安装 npm 依赖后自动执行
    "prepare": "husky install"
  }
}
```

3. 执行

```js
// 为了触发第二步中的prepare
pnpm i
// 或者手动执行
husky install
```

# 添加 git 提交消息规范

1. 安装依赖

```js
pnpm i commitlint @commitlint/cli @commitlint/config-conventional -D
```

2. 创建 commitlint.config.js

3. 添加到 husky

```js
npx husky add .husky/commit-msg "npx --no-install commitlint -e $HUSKY_GIT_PARAMS"
```
